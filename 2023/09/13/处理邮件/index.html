处理邮件

发表于 2023-09-13 更新于 2023-09-26 分类于 python

SMTP

  1. smtplib.SMTP(domain,port)获取stmp对象；
  2. smtp.ehlo()启动与服务器的连接；
  3. smtp.starttls()开启服务端加密，让SMTP连接处于TLS模式；
  4. smtp.login(username,password)使用邮件的账号密码登录邮箱；
  5. smtp.sendmail(from,to,email)发送电子邮件；
  6. smtp.quit()将与邮件服务器断开连接；

IMAP

  1. imapclient.IMAPClient(domain,ssl=True)连接到指定服务器并返回imap对象；
  2. imap.login(username,password)登录到邮件服务器；
  3. imap.select_folder(name,readonly=True)可选择文件夹，imap.search()可在文件夹中搜索邮件；
  4. imap.fetch(UIDS,'BODY[]')可获取邮件并标记已读，imap.delete_messages(UIDS)可删除邮件；
  5. imap.logout()将与邮件服务器断开连接；
