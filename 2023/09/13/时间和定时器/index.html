时间和定时器

发表于 2023-09-13 更新于 2023-09-26 分类于 python

常见的时间函数

  1. time.time()表示当前时刻的时间戳，time.sleep()函数让程序休眠指定时长；
  2. datetime.datetime(year,month,days)可返回指定时刻的datetime对象，datetime.now()返回当前时间的datetime对象，datetime.timedelta(year,month,day,minute,seconds)返回表示一段时间的timedelta对象，如果要实现当前时间前后移多长时间需通过datetime和timedelta相加减实现；
  3. datetime.strftime(format)将datetime转为一个指定格式的时间字符串，datetime.strptime(time_str,format)可将一个字符串转为datetime对象；

多线程、进程

  1. threading.Thread(target = methodName)将创建一个线程对象thread，调用 thread.start()方法即可在新线程中执行methodName所代表的方法；
  2. 调用 thread.join()可阻塞当前线程并等待thread线程运行结束后继续执行；
  3. 调用subprocess.Popen()函数可启动计算机中的其他程序；
