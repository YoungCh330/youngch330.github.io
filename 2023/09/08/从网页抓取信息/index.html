从网页抓取信息

发表于 2023-09-08 更新于 2023-09-26 分类于 python

下载并保存文件

  1. 调用 requests.get()下载该文件；
  2. 利用 response.raise_for_status()校验是否下载成功；
  3. 用’wb’调用 open()，以二进制的方式打开一个新文件，目的是保存文本中的”Unicode”编码；
  4. 利用 response.iter_content()方法做循环，在每个循环中调用 write()方法将文本内容写入文件；
  5. 调用 close()方法关闭文件；

解析HTML

  1. 调用 request.get()获取html文件
  2. 利用 response.raise_for_status()校验是否下载成功；
  3. bs4.BeautifulSoup(response.text,'html.parse')创建bs对象；
  4. bs4.select()查找指定元素，可在chrome浏览器中直接复制selector路径；
  5. 通过 get()获得select返回的Tag对象中各种元素的取值；

使用selenium模块控制浏览器

  1. 使用 webdriver.Chrome()模拟打开浏览器，该方法返回一个浏览器代理对象，注意webdriver needs to be in PATH的问题；
  2. 调用browser代理对象的 get()方法打开指定页面；
  3. 通过 brower.find_element_by_xx()的方法来获取html中的元素WebElement；
  4. 调用WebElement的 send_keys()方法填写内容，调用 click()方法模拟点击，调用 submit()方法提交表单；
