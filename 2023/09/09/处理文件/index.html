处理文件

发表于 2023-09-09 更新于 2023-09-26 分类于 python

读取excel

  1. 调用 openpyxl.load_workbook()函数取得WorkBook对象；
  2. 调用 workbook.active()或sheetnames属性取得WorkSheet对象；
  3. 使用索引或工作表的 cell()方法，带上row和column关键字参数取得Cell对象；
  4. 读取cell的value属性，value会自动转化为对应的整数、字符串、日期类型title；

写入excel

  1. 调用 openpyxl.WorkBook()创建一个空白的WorkBook对象；
  2. 调用 workbook.create_sheet(index=0,title='First Sheet')创建一个WorkSheet对象；
  3. 通过 sheet['A1'] = value为指定单元格赋值；
  4. 调用 del workbook['sheet']删除多余的工作表；
  5. 调用workbook.save(‘Example.xlsx’)保存更改；

其他excel操作

  1. 通过 sheet['A1'].font = Font(name = 'Calibri',bold = True,italic = True)更改单元格的字体；
  2. sheet['B9'] = '=SUM(B1:B8)'可在单元格中添加公式；
  3. sheet.row_dimensions[1].height = 20和 sheet.column_dimensions['B'].wight = 70可设置行高和列宽；
  4. sheet.merge_cells('A1:D3')可合并单元格；sheet.unmerge_cells('A1:D3')可拆分单元格；
  5. sheet.freeze_panes = 'A2'可冻结第一行，sheet.freeze_panes = None可解除冻结；

操作pdf文件

  1. PyPDF2.PdfFileReader(io)将获得PdfReader对象，pdfReader.getPage()可获得指定页面，如果pdf有加密，则可以通过 pdfReader.decrypt('password')解密;
  2. PyPDF2并不能将任意文本写入pdf文档，仅限于从其他PDF中复制页面、旋转页面、重叠页面和加密文件；通过 PyPDF2.PdfFileWriter(io)可创建文件，pdfWriter.addPage(page)可从其他pdf复制页面，page.rotateClockWise(90)可旋转页面，page.mergePage(page)可重叠多个pdf页面以实现水印功能，pdfWriter.encrypt('password')可加密pdf文件；

处理csv文件

  1. open()打开文件，csv.reader(file)可返回csv文件的reader对象，可使用 reader[0][0]来获取指定位置的值，使用 for row in reader来遍历csv的每一行，注意reader对象只能遍历一次；
  2. open(str,'w')打开文件,cvs.writer(file)可返回csv的writer对象，使用 writer.writerRow(list)来逐步写入行，最后调用 file.close()方法结束；
  3. 创建reader和writer对象时，可传入delimiter和lineterminator来指定分隔符和换行符；
  4. 如果需要处理含标题的csv文件，使用DictReader和DictWriter来替代Reader和Writer即可；为避免csv文件中没有标题行，可直接在DictReader第二个参数传入list作为标题；

处理JSON

  1. json.loads(jsonstr)将json字符串转为一个python字典，json.dumps(dict)可将python字典转为json字符串；
